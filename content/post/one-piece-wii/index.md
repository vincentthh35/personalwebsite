前陣子回老家的時候把塵封已久的 wii 再度拿出來玩，也順便把這個好久以前玩到有一定進度了，但卻沒有破完的遊戲拿出來玩。那時候甚至去網路上找了各種攻略印出來，也把遊戲各個場景的地圖也印出來了，跟著攻略打著打著其實也打到蠻後面了，只是之前沒有發現而已。

因為我是撿回來玩所以前面的進度我都忘得差不多了，需要的話可以再去看網路上其他人寫的攻略，唯一的缺陷就是有些攻略有放圖片的，那些圖片大多數都已經過期看不到了，例如地圖上面的釣魚點編號（搭配攻略使用的）等等。

我在撿回來玩之前的進度是到打完青雉左右，隱藏的 boss 只剩下幾個：暴走喬巴、包利、紅頭髮長槍男（卡爾葛拉）、白鬍子，然後也還沒打完魔人，本來跑去興沖沖的打魔人就想說魔人打完就可以開始二回目、打衣服了，可惜我太天真了，要把所有隱藏 boss 也都打完才會開始二回目（我看網路上沒什麼人寫到這部分，應該是因為太基本了）。後來想了一下也覺得蠻合理的，畢竟隱藏 boss 也會有二回目，所以還是得要有一回目解鎖完之後才能打二回目比較合理。這些 boss 被遺留下來沒有被之前的我打完都有一些各自的原因：打包利跟卡爾葛拉都需要抓昆蟲，我印出來的攻略沒有寫昆蟲在哪；暴走喬巴好像是我的水晶球能量不夠，好像缺金色的點數；白鬍子是我根本沒能力釣到黃金鯨魚，而且攻略裡面也沒寫；魔人的話好像是我不知道有這個可以打...。

### 連通道

有些連通道（水晶球事件）雖然說寫在某一個地圖上，但其實不是從那個地圖就能解鎖的，例如在沙濱地圖到密林地圖的那個連通道的事件其實是寫在沙濱那邊，但只能從密林地圖往沙濱走才可以破得到，好像蠻基本的但不知道為什麼我就很有障礙。

密林到沙濱地圖的通道在蠻靠近洞窟到沙濱的通道那邊，只是走另一個方向，走到盡頭之後會跑到沙濱地圖，這邊我也卡超久，我想說光球應該是在草那邊，結果我把草打完之後也沒有跑出光球，走來走去只有一個地上的捕蠅草，用鐵鍬攻擊他也沒有用、站在上面被打也沒用，甚至我想說是不是要有個什麼角色死掉獻祭才會出現（極度愚蠢），最後我不知道怎麼想的就拿有火的道具（好像是火箭或是爆彈）來打那個植物，還真的就是要這樣打掉才有光球出現，出現的那一刻實在是非常感動，我來來回回走了超久...。不知道是不是之前其實有教怎麼處理那種植物，但誰會去打那個...在密林裡面看到都用遠程的打他就可以拿素材了。

天連大地到雪山的連通道也是一樣，在雪山那邊看起來的話是打完紅髮傑克之後的後面應該有一個連通道，但被冰塊擋住了，我到二回目打完青雉的時候才發現原來這個是到天連大地的通道，也是要從天連大地到雪山之後才能解開的這種通道，地點就在青雉競技場的外圍，只要繞一下就可以看到一個有下雪的洞穴，往裡面走就對了。這邊這個解鎖我記得是敲石頭或是地上的叉叉，就不像密林那個那麼有障礙了。

### 開發

開發這邊因為我打 boss 的時候基本上只會用到補血的藥劑，那些炸彈、電池、火箭都不太會用到所以就不會開發，不過這些不會用到的其實很重要，像藍波球（喬巴的 c+甩）是在什麼睡眠炸彈之後才能開發的，因此這些能解然後難度不高的就盡量都開發一下吧，後期很好用蠻強的招，例如：喬巴藍波球、佛朗基人馬、騙人布火鳥星，這些都是要開發完才能用的。

我花最多時間收集開發的是天候棒，但反而也沒啥用，其他上面有寫到的那三招都蠻值得開發的，可以自己斟酌一下，在打 boss 的時候好像沒什麼用娜美瘋狂輸出的印象。

### 抓昆蟲

我覺得打 boss 最麻煩的就是抓昆蟲跟釣魚了，特別是抓昆蟲跟釣魚不一樣的是，釣魚我比較難分辨我是不是在這次釣到我要的魚了，即使失敗也不會覺得很嘔，反而都是釣到了一些奇奇怪怪的魚的時候會想翻白眼而已；昆蟲是他就在那邊，看到了但就是抓不到的那種心情實在是非常痛苦。

昆蟲分成在樹上的甲蟲以及在花上面的蝴蝶，在花上面的蝴蝶基本上就是抓好距離之後由上往下揮網子就可以了（說得簡單，但我在抓彩色蝴蝶的時候也是抓超級多次）；如果是在樹上的甲蟲類的話我覺得用側揮抓比較好抓一點，只要看一下角色的身高就可以有蠻高的機率可以抓中，比起上下揮。反而我遇到的側揮的障礙都是我太用力甩 wii 搖桿了，會讓我左右的判定相反（變成從反方向去敲樹，甲蟲肯定飛走），後來我才發現其實不用很用力揮，輕輕揮其實就能有更精準的判定了。然後也記得如果角色的身高差太多，要去遠一點的地方換角色，不然昆蟲也很容易因為這個而飛走。剩下的就是看運氣跟抓距離的精準度了，玩這個遊戲真的是蠻需要耐心的...。

#### 包利：黃金甲蟲

黃金甲蟲好像是要閱讀一個石碑之後才能抓的，還是怎麼樣其實我不太清楚，不過我看網路上的攻略都說在營地的那個地圖（草原）裡面就有了，在某一個攻略裡面寫說草原的三棵樹小花園那邊有機率出現，其實這邊說的小花園就是在從營地出發過了吊橋之後，到達洞窟地圖之前的那三棵樹，那三棵樹的其中兩棵基本上都會出現一隻甲蟲，如果有看到的話就代表是對的地方，雖說如此黃金甲蟲出現的機率我個人覺得還蠻低的，要在洞窟地圖跟草原地圖之間來回超多次才會看到一次（更別說就算看到了也不一定抓得到）。

如果真的遇到黃金甲蟲的話會蠻明顯的，還記得我在還沒看過黃金甲蟲的時候以為一般甲蟲就是黃金甲蟲，想說都有點黃黃亮亮的，就還是會花時間抓起來。但看到了隻後才知道真的黃金甲蟲會是整隻都金色的，不會像一般的甲蟲頭的部分是黑色的。

後來我覺得繞來繞去發現比較常出現的地方其實是在化石沙濱地圖，從草原的營地出發到沙濱，打完第一波海軍遇到的第一個岔路走右邊，一直走到遇到第二個在溪流邊的岔路也走右邊，那邊會有很多花跟樹，其中會有一棵樹上面有甲蟲，那隻甲蟲我覺得蠻高機率是黃金甲蟲的，不過有個奇怪的地方是在某一棵樹上的甲蟲就算沒有靠很近，只要角色接觸到附近的花也會讓甲蟲飛走，這點我也不太確定原因以及解決辦法（也許是左右繞一下），常常會讓黃金甲蟲就在面前飛走。

抓到了黃金甲蟲之後，這隻 boss 其實沒有很難打，血條也沒有特別厚，只是有一招會把人抓起來扁的那招有點麻煩。

#### 卡爾葛拉：彩色蝴蝶

彩色蝴蝶也是我的心魔之一，網路上的攻略都寫說在天連大地裡面有，但我就是一隻都沒看過，所以很慌，而且天連大地裡面的敵人都強度比較高，又加深我的恐懼，直到我有一次從洞窟到天連大地之後，就看到漂浮在附近的一個島上居然有疑似彩色蝴蝶的昆蟲（真的非常大隻所以很顯眼）。於是就開始看印出來的攻略有沒有寫那個浮島怎麼走，研究了一下發現路線是這樣的：

1. 從洞窟出來之後往 boss 戰的競技場走，有個洞跳進去
2. 出來會到一個荒蕪的浮島，這邊只有一個洞可以跳進去
3. 抵達了有綠意的地區，左邊會有一個黑色石頭搭成的橋，不要管他，往右前方走到盡頭，這時候打開地圖會發現你在兩塊板塊的交界處，這時候就往前跳，就會跳到底下的板塊（都還是在同一個地圖裡面），接著走到底右轉，跳進去那個洞
4. 就會回到洞窟到天連大地的同一個地圖了，但會在不同浮島上，先不要往洞裡面跳，往峭壁走會有一個香菇讓你跳上另一個板塊，一個有湖的板塊，這邊就是目的地了，應該會有幾個敵人打完之後再來抓蝴蝶
5. 抓失敗了沒關係，回到剛剛那個懸崖下的板塊，跳進去那個洞
6. 降落之後再跳進去看到的第一個洞，就會到 3. 附近的一個地方了，就可以循環刷下去了

我看網路上其他攻略有說 2. 那邊的浮島走到底也會有彩色蝴蝶，我沒有證實過但也可以嘗試看看，也許會是一個比較簡單的路線。

抓完蝴蝶之後在 6. 那邊就可以回到草原地圖了，一路往下跳（不是跳到洞裡，而是地圖上的板塊）就可以在同一個地圖抵達連通道。

這隻 boss 我覺得也是數一數二難打的了，他的大招是砸好幾顆鐵球下來，只要砸到基本上就是死掉（除非像佛朗基那種血量怪有機會存活），在砸鐵球的時候可以想辦法躲一下（我是沒想出什麼辦法，都是靠賽），也許滾一下可以躲掉也說不定。近距離攻擊很容易被他抓起來打一頓，遠的攻擊他還有一招會衝刺，是很棘手的 boss。這隻我最後打贏是用暴走喬巴，其實我在第一次觸發喬巴變身的時候還不知道有這個東西，是剛好在吃完藍波球之後死掉才發現，咦怎麼沒死，還跳動畫出來，觸發條件我的猜測是有一定數量成員死掉之後才會觸發這個（讓喬巴太早出去送死好像不會變身），這隻出現之後就可以一路扁 boss 了根本不怕他。

### 採集果實

最難採集的應該就是金色果實了，那時候我還缺的金色水晶球點數可以透過分解金色果實來拿到。至於金色果實出沒的地方就是在雪山地圖裡面的結冰樹上面，有蠻多地方都有結冰樹的，上面都多多少少有機會有，紅髮傑克 boss 戰外面的那一塊區域還蠻多結冰樹的，如果要刷可以在天連大地地圖跟雪山之間來回刷，想當初我沒有把天連大地跟雪山之間的通道打通，只能長途跋涉，非常辛苦，因此建議先把天連大地到雪山的通道打通再來刷比較方便一點。

#### 暴走喬巴

打這個 boss 需要很多耐心，我自己是用騙人布慢慢 A、拉距離、A、拉距離這樣一直循環下去打死的，我覺得跟他肉搏戰沒什麼機會，除非要先準備好很多補藥，看網路上攻略大部分都是寫用魯夫二檔 + 巨人之槍，或者是騙人布，就這樣而已了，我在打二回目的時候騙人布不小心被打死，後來用肉搏戰超難打的，但確實也是用巨人之槍一直扁他才打死的。他的血量真的非常厚，攻擊力也超高，被打到個兩下到三下應該就都要死了，所以要很小心，免得補藥用完了還沒打死。

### 釣魚

釣魚遇到的障礙是釣竿根本沒什麼升級，是到這次回來玩才把釣竿升級到最高級，包含強度以及長度，那些升級其實都有最高級，只是因為需要開發很多次所以才會都看起來沒有變化。比較難收集的我覺得是謎之骨，要在沙濱地圖打完斯摩格之後繼續往下走那邊，地上會有個叉叉印用鐵鍬敲一下就會有了，我一趟都只能拿兩個，不確定其他攻略有沒有比較快的方法。把魚竿升級到最滿了之後整根釣魚竿會變得很不一樣，魚餌的浮標會有黃金梅利號其實還蠻可愛的。其他素材我覺得都沒有特別難拿，可以參考其他攻略看看。

#### 白鬍子：黃金鯨魚

黃金鯨魚是在一個很多海軍的釣場，如果說路線的話從洞窟到沙濱走往海灘之後，右轉會遇到一堆海軍，先不管他們的話左邊會看到一顆岩石，需要衝刺跳躍才跳得過去的地方，跳過去也會有一堆海軍就是那邊了。線的長度我看攻略都寫要在 40m 左右，我自己是沒有很精確的測量過，但確實就是要夠遠才釣得到。我自己在升級釣竿到滿級之後還是不太會把釣魚線弄到那麼長，看了某一個攻略才發現原來是有訣竅的（但我一時之間找不到那個攻略在哪邊）：

1. 按下 A 把右手把指著天空，往正上方舉高
2. 右手把往下讓手臂回到水平之後，往左揮再往右揮動 + 放開 A，左右揮動的間隔越短、力道越大的話釣魚線就會飛越遠

黃金鯨魚就算把釣竿升到最滿級了還是很容易失誤，他的拉扯力道非常強勁，會讓釣魚線耐久度一下就吃掉快半條，如果遇到這種左右拉扯變換很快並且耐久度吃得很快的話，很可能就是釣到黃金鯨魚了。

在釣到之後就可以到天連大地去挑戰白鬍子了，我個人覺得比卡爾葛拉難打一點而已（可見卡爾葛拉多麻煩），一樣難打的地方是他們都有幾乎瞬間秒殺的大絕招，白鬍子是血條比較厚、大絕比較好躲，卡爾葛拉是血條沒那麼長、大絕我不太確定怎麼躲。

至於怎麼走去找白鬍子，這邊有我走法的懶人包：
1. 從洞窟出來之後往 boss 戰的競技場走，有個洞跳進去
2. 出來會到一個荒蕪的浮島，這邊只有一個洞可以跳進去
3. 抵達了有綠意的地區，左邊會有一個黑色石頭搭成的橋，這次要過這個橋了，過橋後走到底跳進洞（這邊會有一個很麻煩的魔人兵，快速通過）
4. 出來了之後會在一個綠地的隆起處，往前走個幾步之後就往左邊的板塊跳下去，順時針繞最大圈就會有一個像是圍著很多刀刃的洞穴，跳進去的下一個地圖就是白鬍子了

打白鬍子我分為兩階段，覺醒前跟覺醒後。第一階段是稍微輕鬆一點的削血階段，這時候的白鬍子只會用兩到三個招數，觀察個幾次應該就可以心領神會了：第一招是站著不動、往前劈砍，這招的距離非常長，只要方向不小心被瞄準就會被扣蠻多血的，有時候跑到很遠的地方想要吃個補血藥都會被爆砍一刀，等於沒補到血還損血；第二招是白鬍子往前跳一小步、用關刀的尾巴敲 + 踩地板（請原諒一個沒看海賊王的人稱呼他武器為關刀），這招要小心的地方是離他背後太近的話也會不小心被踩到，也會損很多血，但好消息是放完這招之後白鬍子會有稍微長一點點的冷卻時間，這個時候就可以打他或是吃藥補血。通常在一開始白鬍子血量還沒有損很多的時候，白鬍子都會用「第一招放三次 + 第二招放一次」的這個循環來攻擊，在第二招放完之後就如前面所說，可以開始打他，打完一波就要見好就收，一波結束之後就盡量不要用後搖太久的招打他（放完之後會在原地不能移動的那種招數，例如：索隆的甩把手），白鬍子只要冷卻完了就會繼續砍下一刀，所以我自己都是用索隆或香吉士接近之後按個 AAA 看情況加個索隆甩搖桿，這樣慢慢消耗白鬍子的血量，想要保險一點的話就 AAA 就好，削血本來就是安全保險一點比較重要。在白鬍子的血量越來越少之後他的攻擊速度會越來越快，這時候要更小心血量會不會不小心整條被打掉，要記得吃藥補血，這時候的白鬍子移動速度超慢也不會衝刺，盡量不要有角色死掉，我自己是會在這時候就開始放香吉士的 c + A，多踢他幾下。

用索隆的小缺點是 AAA 的最後一招距離比較短，而且後搖也比較長，香吉士的話就不太有這個問題，可以一直踢或是移動。也許用魯夫打也不錯，我不用的主要原因只是我覺得第二階段還想用魯夫，所以先留著不用怕不小心讓他死掉了。總之在第一階段就是有耐心一點，盡量不要損血讓角色死掉就可以了。

有個蠻有趣的觀察是在白鬍子覺醒前，從正面打白鬍子他好像比較容易會有「被打退」的動畫（用香吉士 c + A 踢最後一腳之後），在被打退的時候可以繼續再放下一次大絕或是吃藥補血，從後面用香吉士大絕踢他好像就沒有這種反應。而且在覺醒前似乎不能用喬巴的藍波球，我有用了一次然後 wii 就直接卡住當機，也許是我自己機器的問題也說不定，不過依照上面的打法照理來說是不會用到喬巴的。

在打到一定程度之後白鬍子就會覺醒，難度會變成噩夢模式，這時候的白鬍子大致上有三招：第一招是衝刺上劈砍，這招打人超級痛，打到至少損個 1/3 到半血，我自己的觀察是通常在你離他太遠的時候才會用這招（也有可能不是這樣），好在這招他不常用，比較常用的是後兩招；第二招是他會先把關刀拿到頭上揮舞，像直升機一樣，之後開始往面前爆砍超多刀，在砍完叉叉形狀的砍法之後會再接一個往下的垂直劈砍（強烈懷疑是第一階段的第一招），這招也非常可怕，只要在他開始砍之前跑得不夠遠就會全招都吃下來，基本上娜美跟羅賓沒被砍個半死是不可能的，就算是血很厚的角色也沒辦法扛幾次，所以重點就是要在他變直升機的時候趕快跑走，跑走之後看是要補血還是趕快繞到他後面等他砍完之後再開扁都可以；第三招就是大絕了，白鬍子的大絕看起來是一個橫劈產生一個往前飛的劍氣，這招跟直升機是他最常放的兩招，也是幾乎可以秒殺大部分角色的招式。

大絕的話其實在對的時機用 A + B 就可以躲開了，可是還是會有影響到躲招的幾個因素：

1. 手上拿著藥劑或道具（不能 A + B，所以也不建議拿道具去打白鬍子）
2. 角色正在放大招（這個情況好一點，在大招過程中是有減傷的，但還是會受到傷害）
3. 角色放完大絕的後搖（就是站在那邊挨打）
4. 角色放後搖很久的招（大部分角色的甩搖桿我覺得都蠻久的，或者是索隆的 AAA 最後一段）
5. 剛換角色的時候（角色一落地就按 A + B 也可能躲不掉）
6. 距離白鬍子太遠了（距離太遠的躲招時機我覺得不太好抓）

我打下來覺得最常遇到的是 1. 2. 5+6，藥劑或是道具的話可以抓時機等他放完大招也躲掉之後趕快拿藥劑出來吃，吃完之後要趕快收起來免得他又放下一次大招，我自己覺得他在放大招的時候手上還拿著道具的話躲掉的難度又更高（要在一瞬間把道具收起來加上 A + B）；角色正在放大招的話應該是佛朗基跟騙人布比較容易遇到。另外我覺得白鬍子在距離很遠又想放大招的時候（通常是用完直升機那招之後），其實他會思考一下，這時候可以等他思考完並且放大招之後再來吃補藥。

藥劑允許的話我推薦都放大招，這樣比較不會打得那麼辛苦。以下是推薦的幾個大招，都是可以造成肉眼可見傷害的絕招：
* 佛朗基的人馬（c + 甩搖桿）
  * 吃藥的時候要小心白鬍子大招，近身的時候也要注意血量
* 騙人布的火鳥星（c + 甩搖桿）
  * 這個可以打蠻多血的，同樣也是注意吃補藥的時候不要被白鬍子大招偷襲
* 喬巴吃藍波球（c + 甩搖桿，沒能量了就趕快吃補）
  * 這個開了之後不急著送死，光是用 AA 就可以打他很痛了，而且這個狀態下有霸體（幾乎不會被打飛，還可以反過來暴打白鬍子來中斷他的直升機絕招）
  * 因為有霸體所以就一直靠近白鬍子然後 AAAAAAAAA 就可以造成大量傷害
  * 近距離一邊打也要注意他的大招，躲個幾次然後 AA 狂打搞不好也不需要等到暴走喬巴出馬就可以打完了
  * 我自己第一次打贏白鬍子就是用藍波球打的，打死白鬍子還剩下騙人布、魯夫兩個主將
* 魯夫二檔、巨人之槍（c + 甩搖桿、c + A）
  * 這個狀態我覺得跟藍波球的攻擊力可能有得比，都打得很痛，差別在魯夫沒有霸體然後多一個巨人之槍大招
  * 放完巨人之槍之後變小的狀態貌似不能 A + B 翻滾，也不能吃補藥，要注意一下

其他招我覺得都還好，普通攻擊跟大招比起來沒有特別顯著的傷害差別。

> 很搞笑的是我為了要去打白鬍子還特地練了騙人布的火鳥星，來來回回刷到五級了結果第一次打贏根本沒有用到這招。

戰勝白鬍子之後記得注意一下還有沒有剩角色，如果只剩一隻的話就記得把血量補到安全的位置，否則走回去走到一半被殺掉就前功盡棄了。

### 結語

雖然是一個這麼久以前的遊戲，玩起來的畫質也跟現在的其他遊戲機差非常多，但必須說還是一個玩起來非常過癮的作品，特別是對於一個沒什麼看海賊王的人來說，裡面的 boss 大部分我都不知道他們是誰、能力是什麼，玩起來還是可以非常有樂趣，也很有挑戰性，是一個我蠻推薦的遊戲。
